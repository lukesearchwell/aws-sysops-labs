# AWS SysOps Labs

This repository contains a series of hands-on labs focused on developing practical cloud engineering skills using Amazon Web Services (AWS).

### Repository Structure

``` 
  aws-sysops-labs
 │
 ├── ec2-iam-lab
 │   └── notes.md
 │
 └── README.md
```

## Lab Overview

<details>
  <summary> ### EC2 config + IAM Roles </summary>
EC2 + IAM Role Lab
This lab demonstrates secure service access from an EC2 instance using IAM roles.

Topics covered:
Launching EC2 instance via AWS Console
Launching EC2 instance via AWS CLI
Amazon Machine Image (AMI) discovery
IAM role creation and attachment
Instance Metadata Service (IMDSv2)
Retrieval of temporary credentials
S3 access using role-based authentication
IAM permission testing 
Infrastructure teardown

#### Tools Used
- AWS CLI
- Amazon EC2
- AWS IAM
- Amazon S3
- Git
- PowerShell
- SSH
</details>
